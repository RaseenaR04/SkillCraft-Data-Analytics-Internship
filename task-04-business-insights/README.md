## 📌 Project Overview
This project focuses on analyzing marketing campaign performance using Exploratory Data Analysis (EDA).  
The goal is to generate business insights and recommend optimal budget allocation strategies based on performance metrics.

Unlike machine learning projects, this task emphasizes understanding data and extracting actionable insights.

---

## 🎯 Objective
- Analyze marketing campaign data
- Calculate key performance metrics
- Visualize marketing funnel performance
- Compare ROI across channels
- Provide data-driven business recommendations

---

## 📂 Dataset Description
The dataset contains the following columns:

- Channel
- Impressions
- Clicks
- Leads
- Conversions
- Cost
- Revenue

Each row represents the performance of a marketing channel.

---

---

## 📊 Key Metrics Calculated

### 1️⃣ Click Through Rate (CTR)
CTR = (Clicks / Impressions) × 100

### 2️⃣ Conversion Rate
Conversion Rate = (Conversions / Clicks) × 100

### 3️⃣ Return on Investment (ROI)
ROI = ((Revenue − Cost) / Cost) × 100

---

## 📈 Analysis Steps
1. Data Loading
2. Data Understanding using `info()` and `describe()`
3. Missing Value Analysis
4. Metric Calculation (CTR, Conversion Rate, ROI)
5. Funnel Visualization (Impressions → Clicks → Leads → Conversions)
6. ROI Comparison using Bar Chart
7. Business Insight Generation

## 🔎 Key Insights
- Email Marketing achieved the highest ROI.
- Google Ads generated high revenue but had higher acquisition cost.
- Social Media showed high reach but lower conversion efficiency.
- Influencer and Affiliate channels demonstrated moderate performance.

---

## 💡 Business Recommendations
- Increase budget allocation to high ROI channels.
- Optimize Social Media campaigns to improve conversions.
- Maintain Google Ads with cost optimization strategies.
- Monitor influencer campaigns regularly for profitability.

---

## 📌 Conclusion
This analysis demonstrates how Exploratory Data Analysis (EDA) can be used to generate business insights without building predictive models. ROI and conversion metrics play a critical role in making data-driven marketing decisions.
